# Excel Platzi

Repositorio de aprendizaje de Excel desde nivel básico hasta avanzado.

## Estructura
- Básico: fundamentos, fórmulas y visualización
- Intermedio: análisis de datos y productividad
- Avanzado: dashboards y automatización

## Objetivo
Documentar el progreso, ejercicios y notas técnicas adquiridas durante el curso.
