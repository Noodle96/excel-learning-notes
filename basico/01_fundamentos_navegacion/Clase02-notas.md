# Formato visual y presentación de datos en Excel

## Selección rápida de cabeceras
- **Ctrl + Shift + →**: selecciona toda la fila de encabezados.
- Útil antes de aplicar formatos o estilos de celda.

## Estilos de celda
- Inicio → **Estilos de celda**.
- Permiten diferenciar visualmente información clave:
  - Títulos y encabezados
  - Datos
  - Cálculos
  - Incorrecto / Neutral / Modelo
- Ventaja principal: modificar múltiples celdas con el mismo estilo de una sola vez.
- Los estilos pueden **personalizarse** (clic derecho → modificar estilo).

## Personalización de formato
- Grupo **Fuente** (Inicio):
  - Color de celda y de texto
  - Tamaño y tipo de fuente
  - Negrita, cursiva
- Bordes:
  - Aplicar bordes generales o específicos
  - Acceso a opciones avanzadas desde **Más bordes**
- Atajos útiles:
  - **Ctrl + E / Ctrl + A** (según idioma) para seleccionar todo el rango

## Alineación de datos
- Grupo **Alineación**:
  - Centrado horizontal y vertical
  - Rotación de texto
  - Ajustar texto para mostrar contenido largo en varias filas
- Mejora la legibilidad y el orden visual de la información.

## Formato condicional
- Permite resaltar datos automáticamente según reglas.
- Selección rápida del rango: **Ctrl + Shift + ↓**
- Opciones comunes:
  - Conjunto de íconos (semáforos, flechas, círculos)
  - Escalas de color
  - Barras de datos
- Ideal para:
  - Identificar valores altos/bajos
  - Visualizar tendencias
  - Facilitar la toma de decisiones
