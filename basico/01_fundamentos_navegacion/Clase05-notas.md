# Separación de datos y atajos clave en Excel

## Relleno rápido (Flash Fill)
- Permite separar o completar datos siguiendo un patrón.
- Primero escribir el **resultado esperado** en una celda.
- Atajo: **Ctrl + Shift + E** (Excel en español).
- Alternativa: Inicio → Rellenar → Relleno rápido.
- Útil para separar nombres, códigos o texto combinado.

## Texto en columnas
- Ideal cuando los datos están separados por:
  - Espacios
  - Comas
  - Punto y coma
  - Otros delimitadores
- Pasos básicos:
  - Seleccionar rango (**Ctrl + Shift + ↓**).
  - Datos → **Texto en columnas**.
  - Elegir:
    - **Delimitados** (carácter separador).
    - **Ancho fijo** (posición definida).
  - Definir formato de salida (Texto, Fecha, General).
- Convierte una columna en varias correctamente estructuradas.

## Atajos principales de navegación y selección
- **Ctrl + E / Ctrl + A**: seleccionar todo el rango de datos.
- **Ctrl + Shift + \***: selección automática del bloque de datos.
- **Ctrl + flechas**: ir al extremo del rango.
- **Ctrl + Shift + flechas**: seleccionar hasta el extremo.
- Doble clic en borde de columna: autoajustar ancho.

## Atajos de edición y formato
- **Ctrl + 1**: formato de celdas.
- **Ctrl + Z / Ctrl + Y**: deshacer / rehacer.
- **Ctrl + + / Ctrl + -**: insertar o eliminar filas/columnas.
- **Ctrl + F / Ctrl + B**: buscar y reemplazar.

## Formatos rápidos
- **Ctrl + Shift + 4**: formato moneda.
- **Ctrl + Shift + 5**: formato porcentaje.

## Replicación y productividad
- **Ctrl + J**: copiar valor de la celda superior.
- **F4**: repetir la última acción.
- **Ctrl + Enter**: ingresar el mismo valor en varias celdas.
- **Ctrl + Fn + RePág / AvPág**: cambiar entre hojas.
- **Ctrl + F1**: mostrar u ocultar la cinta de opciones.
