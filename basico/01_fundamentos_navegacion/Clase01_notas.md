# Formatos de celda y autorrelleno en Excel

## Tipos de datos en Excel
- **Texto**: se alinea automáticamente a la izquierda.
- **Número / fecha / moneda**: se alinean a la derecha.
- Si una suma no funciona, normalmente el problema es el **formato del dato**.
- En la barra inferior:
  - Suma y promedio → valores numéricos
  - Solo recuento → texto

## Cambiar formato de celda
- Inicio → grupo **Número** para formatos rápidos.
- Clic derecho → **Formato de celdas** para opciones avanzadas.
- Atajo clave: **Ctrl + 1** (abre el menú de formatos).

## Selección rápida de datos
- **Ctrl + Shift + flecha ↓**: selecciona todo el rango con datos.
- Útil para trabajar con tablas largas sin seleccionar manualmente.

## Autorrelleno
- Arrastrar el cuadro verde de la celda.
- Reconoce patrones como:
  - Días y meses
  - Trimestres (enero → abril → julio)
  - Fechas (incluye fin de mes)
  - Texto + número (Q1 → Q2 → Q3)
- Opciones del autorrelleno:
  - Copiar celdas
  - Rellenar serie
  - Copiar solo valores o formatos

## Buscar y reemplazar
- Atajo: **Ctrl + L**
- Permite reemplazos masivos de datos.
- Útil para limpieza rápida de información.
