# Funciones matemáticas básicas en Excel

## Estructura de una función
- Toda función comienza con **=**.
- La función se muestra en **negrita** mientras se escribe.
- Los argumentos se separan con **coma (,)**.
- Los elementos entre **corchetes [ ]** son opcionales.
- Los **puntos suspensivos (...)** indican que el argumento puede repetirse.

## Funciones esenciales para análisis financiero

### SUMA
- Calcula el total de un rango.
- Uso común: total de ingresos o gastos.
- Ejemplo:
  - `=SUMA(A1:A10)`

### PROMEDIO
- Calcula el valor medio de un rango.
- Útil para analizar ingresos promedio.
- Ejemplo:
  - `=PROMEDIO(A1:A10)`

### MAX
- Devuelve el valor más alto del rango.
- Útil para identificar el mayor ingreso o gasto.
- Ejemplo:
  - `=MAX(A1:A10)`

### MIN
- Devuelve el valor más bajo del rango.
- Permite detectar mínimos financieros.
- Ejemplo:
  - `=MIN(A1:A10)`

## Funciones para conteo de datos

### CONTAR
- Cuenta solo celdas con valores numéricos.
- Ideal para contar transacciones con importe.

### CONTARA
- Cuenta celdas con cualquier tipo de dato.
- Útil para validar registros completos.

## Atajos útiles
- **Ctrl + Shift + ↓**: seleccionar un rango de datos.
- **Ctrl + Shift + 4**: aplicar formato de moneda.
